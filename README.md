# Leksjon 6 · Lesetekster i farger

GitHub-ready lesson file for self-study.

## Structure

- `index.html` - main lesson page
- `audio/01_Oslo.mp3` - text 1 audio
- `audio/02_Bergen.mp3` - text 2 audio
- `audio/03_Tromse.mp3` - text 3 audio

## Use on GitHub Pages

Upload the whole folder contents to a repository. Enable GitHub Pages from the repository settings and use `index.html` as the start page.
